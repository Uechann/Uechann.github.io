# PRD-07 — 커스텀 관리자 페이지

> 상태: 🔴 미구현  
> 우선순위: 높음

---

## 목표

Sveltia CMS 제거, 블로그 디자인과 완전히 동일한 커스텀 어드민 페이지를 직접 구현한다.  
GitHub API + PAT 인증으로 파일을 직접 읽고/쓴다.

---

## 구현 방식

```
static/admin/
└── index.html   ← SPA (HTML + CSS + JS 단일 파일)
    config.yml   ← 삭제 (더 이상 불필요)
```

외부 의존성: EasyMDE (CDN, 마크다운 에디터) 하나만.

---

## 화면 구성 (SPA Views)

### 로그인
- PAT 입력 + 토큰 발급 안내

### 대시보드
- 사이드바 카테고리 네비게이션
- 카테고리별 글 수 카드
- 빠른 글쓰기 버튼

### 카테고리 글 목록
- 제목, 작성일, 초안/발행 상태
- 편집 버튼

### 글 에디터
- 제목 (큰 입력 필드)
- URL Slug (새 글만)
- 작성일, 초안 토글, 카테고리 멀티셀렉트, 태그
- 요약, 커버 이미지, 시리즈명
- EasyMDE 마크다운 에디터
- 초안 저장 / 발행 / 삭제

### 소개 페이지 편집
- 제목 + EasyMDE 에디터

---

## 디자인 스펙

- 폰트: Pretendard (블로그와 동일)
- Accent: #3b82f6
- 다크모드: CSS 변수 + prefers-color-scheme
- 사이드바 220px 고정, 카테고리별 색상 dot
